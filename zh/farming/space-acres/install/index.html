安装

开始之前​

确保您具备以下条件：

  1. 一台具有可靠互联网连接的计算机。
  2. 足够的可用磁盘空间 (我们稍后会帮助您确定所需空间)。
  3. 拥有计算机的管理员访问权限。

会发生什么？

  • 同步可能需要几小时到一天的时间。
  • 初始同步和扇区绘制过程消耗大量计算资源，因此您可能会注意到在同步和绘制完成之前，计算机的日常任务运行缓慢。
备注

请注意，运行该软件需要一个具有Substrate地址的钱包。
要了解如何创建Substrate钱包，请访问我们关于 SubWallet(推荐) 或 {polkadot.js} 扩展

端口转发

为确保 Space Acres与其他节点间正常通信， 您需要将路由器中的 30333 和 30433 端口转发到运行 Space Acres的 PC 。 欲了解更多详情，请参阅端口转发指南。

安装视频​

对于那些喜欢视频类指南的用户，我们已为您准备了安装视频。这些视频将帮助您设置Space Acres。如果您更喜欢详细的文字说明，请继续查看 步骤1。

步骤1：下载并安装Space Acres​

  • 🐧 Linux
  • 🖼️ Windows
  • 🍎 macOS

下载Space Acres安装程序或便携应用​

目前，有两种方法可以在Linux上安装Space Acres：

  • 安装Ubuntu的.deb包(推荐)
  • 直接在任何现代Linux发行版上运行.AppImage捆绑包(包括Ubuntu，如果您愿意的话)
Ubuntu Installer(.deb)Linux Portable App(AppImage)

安装Space Acres​

您可以使用提供的 (.deb)安装程序安装应用程序，或运行便携应用程序 (.Appimage)。

Ubuntu安装程序 (.deb) (推荐)​

  1. To install the app, open terminal and run the following commands to switch to downloads directory and install an app

    cd Downloads
    sudo apt install ./space-acres*.deb

    Replace Downloads with the correct name of downloads directory if you have non-English Ubuntu installation or if you downloaded file into a custom location. In case you have multiple versions of Space Acres downloaded, you might want to replace space-acres*.deb with a full name of the file you've downloaded. Or go to Downloads directory and double-click the application to start the installation process

  2. 点击屏幕左下角的“显示应用”图标。

  3. 前往Space Acres应用程序（如果您安装了很多应用，可能需要滚动鼠标才能找到它所在的页面）。

  4. 双击Space Acres，应用程序将启动。

便携应用(.AppImage)​

  1. 打开文“File”（文件）应用程序。

  2. 点击打开“Downloads”（下载）文件夹。

  3. 右键点击 .AppImage 文件并选择“Properties”（属性）。

  4. 将“Executable as program”选项切换为开启状态。

  5. 关闭Properties（属性）窗口。

  6. 双击 .AppImage，Space Acres 将启动。

下载Space Acres安装程序​

Windows Installer(.exe)

安装Space Acres​

  1. 打开下载的 Space Acres 安装程序以启动设置向导。此向导将指导您完成安装过程。

  2. 选择您的安装偏好，例如目标文件夹和是否创建快捷方式。您还可以选择在电脑启动时启动 Space Acres。

  3. 查看您的设置，然后点击安装开始安装。向导将在您的系统上安装Space Acres。

  4. 安装进度会有显示，展现正在复制的新文件的状态。

  5. 一旦安装完成，您将看到一个完成屏幕，在此您可以选择立即运行 Space Acres，只需点击“完成”之前勾选立即运行。

下载Space Acres安装程序​

macOS Installer(.dmg)

安装Space Acres​

  1. 通过双击打开下载的Space Acres安装程序。

  2. 将Space Acres拖入您的“应用程序”文件夹。

  3. Launch the application upon successful installation.

步骤2：设置Space Acres​

现在进入激动人心的部分——设置您的农场！以下是您在第一个屏幕上看到的内容：

  1. Node path:

    • 这将是Space Acres存储重要数据的地方。
    • Pick a location with plenty of free space; it might be as simple as /Documents.

  2. Rewards address:

    • 这是您的收益耕作奖励将被发送到的地址。
    • It starts with su for SubWallet and with 5 for {Polkadot.js} extension.
    • 如果您还没有任何地址，请点击“Create wallet”来创建一个钱包。

  3. Path to Farm 0 and its size:

    • 这是您的 **存储扇区 **(存储收益耕作数据) 位置。
    • 存储扇区的大小决定了您获得奖励的概率。在大多数情况下，更多的扇区(或较大的农场) 将为您带来更多奖励。
    • The size could be set as a fixed size, e.g. 4TB, or 500GB, or as a percentage value of the available disk space.
    • 如果您希望拥有几个农场(例如，您在不同的SSD驱动器上有一些闲置空间)，可以点击“Add farm”来添加额外的存储空间。
    提示

    The screen above is just an example of configured setup screen, make sure to use your own Node path, Rewards address, Path to farm 0 and its size, and Fixed size / Disk space percentage values

  4. 高级配置(可选):

    • Toggle Reduce plotting CPU load (select this option if you're using your farming PC for other tasks)
    • Modify the default values for blockchain and DSN P2P ports
    • Toggle Faster Networking (selecting this option may increase the load on your router)
    提示

    如果您不是资深用户，请不要担心这个。

    关于前提条件和说明的全面清单，请参阅我们的开始指南.

    信息

    Your screen may look slightly different depending on your operating system, but the basic information required will be the same.

步骤3：开始收益耕作​

  1. 输入您的信息后，点击Start。
  2. Space Acres将设置您的节点并连接到Autonomys网络。
  3. 您将看到进度条，显示数据同步和创建您的存储扇区的过程。

  1. 一切准备就绪后，您将看到Space Acres Dashboard，您可以监控您的收益耕作活动。

  2. 初始同步和存储扇区绘制可能需要从几小时到几天，具体取决于您的硬件、网络速度和指定的农场大小。以下是您可以预期的各个阶段的示例。

    a. 缓存同步

    b. 初始存储扇区绘制

    c. 存储扇区绘制完成，现在开始收益耕作

  3. Once syncing and plotting are complete, you'll notice a drop in CPU and RAM usage, and the dashboard will display your earned rewards and the estimated time until the next reward.

    提示

    You can toggle the dashboard to view brief statistics or display all plotted sectors along with their status. The color codes of sectors are explained below. By hovering over the icons on the right, you can check the number of missed challenges, your average auditing and proving performance, and other useful stats.

Happy farming!​
