Polkadot

备注

polkadot.js插件是 Substrate/EVM 钱包，由 Substrate 和 Polkadot/Kusama 的开发者 [Parity Team] (https://www.parity.io/) 创建。

这是一款极简钱包。 This is because it is a wallet that supports all Substrate based networks. 一个类似于Metamask的插件，或您过去可能使用过的其他基于浏览器的钱包。

Create a New Wallet​

  1. 访问 Polkadot.js 网站并下载相应版本。
提示

Chrome选项适用于所有基于 Chromium 的浏览器，如Brave、Vivaldi和Edge

  1. 安装插件后，打开插件。 阅读注释。

  2. 点击 + 添加一个新帐户。

  3. 然后，插件将向你显示 12 个单词的助记词。

危险

务必安全存储，切勿与他人共享

  1. Once your seed is securely stored and saved, click the “I have saved my mnemonic seed safely” check box and click “Next Step”

  2. 下一步将询问钱包的名称和密码。 然后单击“Add the account with the generated seed”

  3. 恭喜你已创建 polkadot.JS 钱包！

导入现有助记词​

某些用户可能已经有Subspace-Desktop 提供的助记词，如果是这种情况，你可以按照指南的这一部分操作。

  1. 安装插件 (请参阅步骤1 之前的部分)

  2. 一旦安装插件，打开它并点击 + 和 Import account from pre-existing seed

  3. 键入或粘贴到您的12个单词的助记词，然后单击 Next

  4. 下一步将询问钱包的名称和密码。 然后单击 Add the account with the supplied seed

Troubleshooting​

如果您遇到任何问题或想了解关于Polkadot.js的其它功能，请查看Polkadot.js 官方文档。 我们在下面列出了一些基本的常见问题解答。

我怎样才能找到我的公钥？​

  • 你可以在插件内你的钱包名称正下方看到你的默认substrate公钥

  • You can see your Autonomys mainnet public address via the ... menu and setting the Allow Use on Any Chain dropdown to Autonomys mainnet, once you exit you will see the public address now starts with su

I Don't See Autonomys mainnet or Any Autonomys Networks as an Option in Chain Settings​

  • 如下所示，有时当你首次安装或更新Substrate钱包时，你需要更新钱包元数据。

  1. Go to the Subspace/Polkadot Explorer here: Polkadot/Substrate Portal

  2. You will be prompted to allow the extension to connect, Select desired account(s),click Connect 1 account(s)

  3. 在网页上点击 settings

  4. 点击 Metadata

  5. 单击Update Metadata

  6. 插件将弹出一个窗口，请您确认点击Yes, do this metadata update

  7. You will now see Autonomys mainnet as an option on the Allow use on any chain dropdown.

How Do I Backup my Wallet?​

  1. 您可以通过...菜单，点击 Export Account 备份/导出钱包

  2. 然后输入您的钱包密码，单击 I want to export this account
