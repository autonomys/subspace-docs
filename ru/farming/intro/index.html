Приступая к работе

Видео по установке

Для тех, кто предпочитает наглядное руководство, мы подготовили видеоролики по установке. Видео помогут вам настроить фермерское приложение Space Acres Recommended менее чем за 10 минут. Пожалуйста, следуйте видеороликам установки, чтобы использовать SubWallet для создания совместимого аккаунта Substrate и обратиться к подробному руководству по загрузке и настройке фермерского приложения Autonomys, Space Acres.

Замечание об интернете

Работа узла и фармера может быть весьма интенсивной. Рекомендуется иметь стабильное и неограниченное интернет-соединение.

Эталонное оборудование​

Наши тесты на бенчмаркинг были проведены с использованием процессора Intel Core i7-6700. Мы калибровали вес транзакции, чтобы убедиться, что полный блок не содержит больше транзакций, чем этот процессор может обрабатывать в течение 2-х секунд ограничения времени выполнения транзакций. Это гарантирует, что узлы с аппаратным обеспечением, эквивалентным или лучше, чем процессор Intel Core i7-6700 может блокироваться при полной мощности без проблем с производительностью.

Хотя фармеры, использующие аппаратные средства ниже этой спецификации, могут эффективно функционировать в большинстве условий, они могут столкнуться с трудностями во время пиковой сетевой активности. Если блоки постоянно заполняют транзакции, узлы с меньшими спецификациями могут бороться за сохранение темпа, что может повлиять на производительность фарминга и вознаграждение.

Вы можете увидеть подробную информацию о машине, используемую для эталонного оборудования.

Хранилище​

Жесткие диски не поддерживаются

Жесткие диски не поддерживаются. Использование интеллектуального кэширования, многоуровневого хранения, RAID или других методов ускорения будет неэффективным из-за значительного количества операций случайного чтения IOPS, необходимого для процесса доказательства.

Требуется SSD-накопитель. Лучшие модели не обязательны, достаточно SSD среднего класса от авторитетного производителя с достаточным запасом прочности. Будьте осторожны с менее известными брендами, поскольку у них могут возникнуть проблемы с поддержанием стабильной производительности чтения. Можно также использовать твердотельные накопители NVMe, но это не обязательно.

Файловая система (Linux)​

Лимит дескрипторов файлов Linux

В системах Linux может быть установлен лимит файловых дескрипторов по умолчанию, который может меняться в зависимости от дистрибутива. Превышение этого лимита может привести к ошибкам. Для получения подробной информации посетите наше руководство Tips & Tricks.

  • ext4
  • XFS
  • CoW
  • RAID
  • SWAP

Эта файловая система максимизирует пространство для фармера.

Recommended Formatting Command for Drives Dedicated to Subspace Farms:
sudo mkfs.ext4 -m 0 -T largefile4 -C 16384 -O bigalloc /dev/sdX

Изменение размера фермы при почти полном заполнении диска (98 % или более) приведет к тому, что ферма станет нерабочей, если не будет освобождено достаточное количество дополнительного пространства другими способами. Полная загрузка диска может быть достигнута только при первоначальном создании фермы и даст меньше полезного пространства по сравнению с ext4.

Файловые системы Copy-on-Write (ReFS, BTRFS, ZFS, bcachefs, HAMMER) не рекомендуется из-за увеличения задержки и уменьшения скорости записи связанных с управлением ненужными копиями.

Disable CoW on BTRFS:
sudo chattr +C /path/to/farm

Фермер использует контрольные суммы для обеспечения целостности данных и может восстанавливать поврежденные блоки. Создание рейда не дает никаких преимуществ и повышает риск необходимости перераспределения данных по нескольким дискам вместо одного.

Необходим только если ваша система имеет недостаточно ОЗУ.

Обьяснение блокчейна​

Узнайте больше о сети Autonomys с помощью наших различных средств телеметрии и обозреватели блоков. Независимо от того, отслеживаете ли вы сетевую активность или изучаете данные блокчейна, эти ресурсы обеспечивают всестороннее понимание производительности и транзакций в сети.

  • Сервер телеметрии: Получайте информацию в реальном времени о сетевой активности и показателях производительности. Идеально подходит для мониторинга общего состояния и состояния сети Autonomys.

  • Официальный обозреватель блоков - Astral: Наш основной инструмент для просмотра блоков, транзакций и сетевой активности в сети Autonomys. Этот обозреватель предлагает интуитивно понятный интерфейс и подробную информацию.

  • Обозреватель блоков - Subscan: альтернативный обозреватель блоков, обеспечивающий подробное представление блоков, транзакций и сетевых событий. Subscan известен своим удобным интерфейсом и дополнительными функциями аналитики данных.

  • Polkadot.js Block Explorer: Для пользователей, знакомых с экосистемой Polkadot, этот проводник предлагает беспрепятственный опыт для изучения сети Autonomys Network с помощью интерфейса Polkadot.js.

Зависимости​

Проблема с отсутствием вывода в Windows

Если вы столкнулись с ошибкой, при которой нода ничего не выводит и не указан код ошибки, то достаточно просто установить последний пакет Visual C++ Redistributable здесь

Сеть​

Проброс портов и Руководство брандмауэра

Для получения подробной информации о конфигурации сети, включая требования к перенаправлению портов и настройки брандмауэра, обратитесь к руководству Перенаправление портов и брандмауэр.

Рекомендации по безопасности​

Защита и безопасность

Для получения подробной информации посетите наше руководство Безопасность и безопасность.

Убедитесь, что ваша система регулярно обновляется, настройте проброс портов, настройте свою firewall и следуйте наилучшим критериям безопасности сети. Для получения подробных инструкций по безопасности смотрите руководство Безопасность и безопасность.

Кошелек​

Руководство по кошелькам

Для получения подробной информации посетите наш гид по SubWallet или Polkadot.JS.

Прежде чем начать, убедитесь, что у вас есть кошелек для получения наград. Мы рекомендуем использовать SubWallet, но [PolkadotJS](https://polkadot.js. Также доступен rg/extension). Установите один из этих кошельков в качестве расширения для браузера и создайте новую учетную запись. Адрес вашей учетной записи будет необходим в процессе установки.
