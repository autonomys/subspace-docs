This guide explains how to safely securely transfer and bridge tokens between Auto EVM and supported external blockchain networks.

### Supported Networks

The Autonomys Bridge currently supports the following external networks:

| Network   | Status |
| --------- | ------ |
| Ethereum  | Live   |
| BNB Chain | Live   |
| Solana    | Live   |

:::tip
The step-by-step instructions in this guide utilize Ethereum as the baseline example, but the exact same transactional flow applies to BNB Chain and Solana. Simply select your preferred network interface within the bridge UI when choosing your source or destination chain.
:::

### Connecting Your Wallet to the Bridge

:::note Mainnet and Testnet Environments

- **[Mainnet Bridge Portal](https://bridge.mainnet.autonomys.xyz/)**
- **[Chronos Testnet Bridge Portal](https://bridge.chronos.autonomys.xyz/)** — Use this environment to test bridging operations securely with testnet assets.

To acquire `tAI3` testnet tokens, utilize the [Official Web Faucet](https://autonomysfaucet.xyz/) or request them via the `/faucet` slash command within the dedicated `#faucet` channel on the [Autonomys Discord Server](https://autonomys.xyz/discord). Token requests are limited to once every 24 hours. Review the [Full Faucet Guide](https://develop.autonomys.xyz/evm/faucet) for deeper technical parameters.
:::

1. Navigate to the official [Mainnet Bridge Platform](https://bridge.mainnet.autonomys.xyz/).

![Bridge](/img/doc-imgs/additional-guides/bridge-01.jpeg)

2. Connect your Web3 wallet by clicking the `Connect Wallet` action button located in the upper right-hand corner of the user interface.

![Bridge](/img/doc-imgs/additional-guides/bridge-02.jpeg)

3. You will be prompted to select your preferred provider. This guide demonstrates the flow utilizing MetaMask.

![Bridge](/img/doc-imgs/additional-guides/bridge-03.jpeg)

4. Authorize the connection profile within the MetaMask extension popup.

![Bridge](/img/doc-imgs/additional-guides/bridge-04.jpeg)

5. Once authorized, your active wallet address will display in the top right corner. Note that the recipient field automatically defaults to your currently connected wallet address.

![Bridge](/img/doc-imgs/additional-guides/bridge-05.jpeg)

## Tracking Bridge Transactions

Click the clock icon on the application interface to view historical bridge ledgers and verify past transaction executions. To close the history slide-out panel, simply select the navigation arrows located on the left edge.
