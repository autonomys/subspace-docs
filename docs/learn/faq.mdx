### How do you change the default ports? {#change-default-ports}

ğŸ“ If you need to change the default ports for the farmer and node, whether to run multiple instances on the same local network or to operate both the mainnet and the Taurus testnet at the same time, you must configure them to use unique ports.

<details>
<summary>CLI</summary>

Use the following parameters to assign new ports, ensuring they do not conflict with existing ones on the network:

<h2>Node</h2>
```shell
listen-on "/ip4/0.0.0.0/tcp/30333"  
dsn-listen-on "/ip4/0.0.0.0/tcp/30433"  
```

<h2>Farmer</h2>
```shell
listen-on "/ip4/0.0.0.0/tcp/30533"  
```
</details>

<details>
<summary>Space Acres</summary>

Open the **Hamburger Menu**, select **Change configuration**, then go to **Advanced configuration** to modify the port settings.
</details>
For more details, including steps to enable port forwarding, refer to the [Ports to Forward & Firewall Access](/farming/guides/port-config) guide.

### How long will it take to plot my farm? {#plotting-time}
ğŸ“ The initial plotting time depends on your hardware. Using a [supported GPU](/farming/guides/gpu-plotter) can significantly speed up the plotting process while consuming less power. However, CPU plotting is also supported if you prefer or don't have access to a compatible GPU. Once the initial plotting is complete, ongoing replotting (as sectors expire) will take less time. You can check out the [plotting speed results](https://ai3.farm/resources/plotting-speed) from community members to get an idea of what to expect based on different hardware setups.
